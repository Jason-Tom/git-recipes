如何修改eclipse的workspace目录

百度经验:jingyan.baidu.com

在第一次启动eclipse时，会弹出Workspace Launcher对话框提示设定workspace目录，如果我们将“Use this as the default and do not ask again”这个选项勾选，之后再登录将不再提示目录设定，那么我们如果想要修改workspace目录该如何去做

百度经验:jingyan.baidu.com

工具/原料

  • eclipse
百度经验:jingyan.baidu.com

方法/步骤

  1. 1

    打开Window，选择Preferences-》General--》Startup and Shutdown-》Workspaces，勾选Prompt for workspace on startup 选项

    步骤阅读
  2. 2

    上述设定设置好之后，关闭eclipse，重新启动时便会弹出Workspace launcher对话框

    根据需要修改workspace目录

    步骤阅读
  3. 3

    还有一种简单方案就是，eclipse启动状态下，再启动一次，这时会提示workspace目录正在使用中，点击【OK】，也会弹出Workspace launcher对话框，按需要修改Workspace 目录
